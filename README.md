# Roman Numeral Calculator

A modular Python application that converts Roman numerals to integers,
performs arithmetic operations, and returns results in Roman numeral
format.

This project demonstrates structured programming, input validation,
error handling, and algorithmic problem solving using Python.

------------------------------------------------------------------------

## Features

-   Convert Roman numerals to integers\
-   Convert integers back to Roman numerals\
-   Perform arithmetic operations:
    -   Addition (+)
    -   Subtraction (-)
    -   Multiplication (\*)
    -   Division (/) with quotient and remainder
    -   Exponentiation (\^)
-   Input validation for Roman numerals\
-   Range enforcement (1--3999)\
-   Modular function-based design

------------------------------------------------------------------------

## How It Works

1.  The user enters two Roman numerals.
2.  The program converts them to integers.
3.  The user selects an arithmetic operation.
4.  The operation is performed.
5.  The result is validated to ensure it falls within the accepted Roman
    numeral range (1--3999).
6.  The result is converted back to Roman numerals and displayed.

For division operations, both the quotient and remainder are returned in
Roman numeral format.
